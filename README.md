# DSA-AI-ML-Project

## PROJECT OVERVIEW

This project involved the development of an end-to-end credit card fraud detection system that analyzes transaction behavior and predicts whether a transaction is fraudulent or legitimate. The system combines exploratory data analysis, machine learning modeling, and web deployment, resulting in a fully functional and publicly accessible application.
